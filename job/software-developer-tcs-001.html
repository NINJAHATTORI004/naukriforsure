Software Developer

Tata Consultancy Services (TCS)
Mumbai, Bangalore, Pune Full-time Posted 2 days ago

Job Description

We are looking for passionate freshers to join our team as Software Developers. As part of TCS, you'll work on cutting-edge technology projects for global clients.

This is an excellent opportunity for fresh graduates to kickstart their career in IT with one of the world's leading technology consulting companies.

Key Responsibilities

  • Develop and maintain software applications using Java, Python, or JavaScript
  • Participate in code reviews and maintain coding standards
  • Collaborate with cross-functional teams to define and implement new features
  • Write clean, scalable, and well-documented code
  • Debug and troubleshoot software issues
  • Participate in Agile development processes

Requirements

  • B.E/B.Tech/MCA or equivalent degree in Computer Science or related field
  • Strong understanding of programming fundamentals
  • Knowledge of at least one programming language (Java, Python, C++)
  • Good problem-solving and analytical skills
  • Excellent communication skills
  • Willingness to learn and adapt to new technologies

Skills Required

Java Python SQL JavaScript Problem Solving Communication

Benefits

  • Competitive salary with annual increments
  • Health insurance coverage
  • Paid time off and holidays
  • Learning and development opportunities
  • Career growth within TCS
  • Work-life balance initiatives
