Data Analyst

Wipro Limited
Bangalore, Delhi NCR Full-time Posted 1 day ago

Job Description

Wipro is looking for Data Analysts to join our growing analytics team. Work with large datasets to derive meaningful insights and support data-driven decision making for our clients.

This is an excellent opportunity for freshers passionate about data analysis and visualization.

Key Responsibilities

  • Collect, process, and analyze large datasets
  • Create visualizations and dashboards using tools like Tableau/Power BI
  • Write SQL queries for data extraction and manipulation
  • Identify trends, patterns, and anomalies in data
  • Prepare reports and presentations for stakeholders
  • Collaborate with teams to understand data requirements

Requirements

  • B.E/B.Tech/B.Sc/M.Sc in Computer Science, Statistics, or Mathematics
  • Strong analytical and statistical skills
  • Proficiency in SQL and Excel
  • Knowledge of Python/R for data analysis is a plus
  • Good visualization skills
  • Attention to detail and accuracy

Skills Required

SQL Excel Python Tableau Statistics Data Visualization

Benefits

  • Competitive salary package
  • Medical and life insurance
  • Learning allowances
  • Flexible work arrangements
  • Career progression opportunities
