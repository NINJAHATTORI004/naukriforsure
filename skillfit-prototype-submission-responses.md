### 2. Description *
SkillFit Bharat AI is a revolutionary multilingual, mobile-first AI hiring and workforce fitment platform designed specifically for India's underserved talent pool. 

**What it does:**
- Conducts AI-powered video and voice interviews in local languages (Kannada, Hindi, English, expandable to Tamil, Telugu, Marathi, Bengali)
- Verifies candidate authenticity through face detection, liveness checks, and behavioral analysis
- Evaluates employability across 7+ dimensions: communication clarity, role-fit knowledge, confidence, professionalism, skill relevance, consistency, and availability
- Detects fraud and duplicate applications with ML-powered algorithms
- Matches job-ready candidates to employers, apprenticeships, government programs, and training centers in minutes

**Key Innovation - Digital Skill Passport:**
Every candidate receives a reusable, AI-generated employability profile containing verified identity confidence, communication score, role readiness, language capability, reliability score, recommended job families, and suggested training modules.

**Impact:**
- Millions of blue-collar, semi-skilled, and ITI candidates gain access to formal employment
- Governments and employers identify job-ready talent at scale without resume dependencies
- Eliminates English fluency barriers and geographic constraints
- Reduces hiring time from weeks to minutes

**Revenue Model:** Government SaaS licensing, enterprise per-candidate fees, training partnerships, and verified talent marketplace subscriptions.

### 11. Instructions to Run *

#### Prerequisites:
- Node.js v16+ or Python 3.8+
- Docker & Docker Compose (recommended)
- MongoDB (for data storage)
- Redis (for caching)

#### Local Setup (Docker - Recommended):

```bash
# Clone repository
git clone https://github.com/NINJAHATTORI004/skillfit-bharat-ai.git
cd skillfit-bharat-ai

# Copy environment files
cp .env.example .env

# Start all services with Docker Compose
docker-compose up -d

# Run database migrations
docker-compose exec backend npm run migrate

# Seed sample data
docker-compose exec backend npm run seed

# Access the application:
# - Candidate App: http://localhost:3000
# - Admin Dashboard: http://localhost:3001
# - API Documentation: http://localhost:5000/api/docs
```

#### Manual Setup:

```bash
# Backend Setup
cd backend
npm install
npm run dev

# Frontend Setup (in another terminal)
cd frontend/admin-dashboard
npm install
npm start

# Candidate Mobile App (React Native)
cd frontend/candidate-app
npm install
npm start
```

#### Testing the Prototype:

1. **Candidate Registration:**
   - Navigate to http://localhost:3000
   - Click "Get Started"
   - Enter details: name, mobile, language (select Kannada/Hindi/English)
   - Select skill area, location, experience level

2. **Mock Interview:**
   - Click "Start Interview"
   - View AI questions (currently mock data)
   - Record or type responses (simulated for prototype)

3. **View Results:**
   - See evaluation scores
   - View Digital Skill Passport
   - Check job recommendations

4. **Admin Dashboard:**
   - Navigate to http://localhost:3001
   - Login: admin@skillfit.com / demo123
   - Filter by district, skill, readiness level
   - View analytics and insights

#### API Endpoints (for testing):

```
POST /api/auth/register - Candidate registration
POST /api/interview/start - Start interview session
POST /api/interview/submit - Submit interview responses
GET /api/passport/:candidateId - Get Digital Skill Passport
GET /api/jobs/recommendations/:candidateId - Get job matches
GET /api/admin/dashboard - Admin dashboard data
GET /api/admin/candidates - List candidates with filters
```

#### Troubleshooting:

- **Port already in use:** Change ports in `.env` file
- **Database connection error:** Ensure MongoDB is running
- **ML models not loading:** Check MODELS_PATH in environment variables
- **CORS errors:** Verify ALLOWED_ORIGINS in backend config
